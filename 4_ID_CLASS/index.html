ID e CLASS

ID

o ID funciona como um CPF. Assim como so posso ter um numero de CPF, cada elemento so pode ter um valor de ID. O CPF e unico, ou seja ninguem pode ter um CPF igual ao meu. Isso significa que o ID tambem e unico, entao e possivel repetir o mesmo valor deuas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que voce so pode ter um valor, no CLASS voce pode ter varios valores. tambem diferente do ID, em que nao se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor para varios elementos.
